# Red Tara Sanctuary

Static guest guide website for Red Tara Sanctuary in Catskill, NY.

## Files

- `index.html` - guest-facing guide
- `styles.css` - minimal responsive styling
- `airbnb-listing-suggestions.md` - owner-facing Airbnb listing copy and checklist

## GitHub Pages

This site is built as plain static HTML/CSS and can be served directly from the repository root with GitHub Pages.

Private access details such as door codes and Wi-Fi passwords should not be committed to this public repository.
